Para formatar o hino que estará no arquivo 'songs.txt', é necessário utilizar algums marcadores.

 * Marcador - => Insere uma tag <p>
 * Marcador + => Insere uma tag </p>
 * Marcador [ => Insere uma tag <kbd>
 * Marcador ] => Insere uma tag </kbd>
 * Marcador ( => Insere uma tag <pre>
 * Marcador ) => Insere uma tag </pre>

Exemplo:

 - Cifras:
    ( [G] [C] [Am])


 - Texto:
    -Cristo já nos preparou, um manjar que nos comprou+


Observação: As quebras de linha são importantes, ou seja, após as cifras, deverá haver um 'enter' para iniciar o texto.
